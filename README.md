Welcome to the Ghost of Yōtei Database

This is a community-made database for Ghost of Yōtei and is not officially affiliated with Sucker Punch Productions.

This site was created by @rrsmirnoff on Discord. The goal of this project is to provide:

patch notes
builds
guides
weapon information
class setups
and general game information

Everything on this site is researched through patch notes, in-game testing, and personal gameplay experience.

If you notice any mistakes, outdated information, or missing content, feel free to message me on Discord with corrections or suggestions. I’ll do my best to review and update the site within 24 hours.

This project was built and maintained by one person, so I appreciate all feedback and support from the community.

Thank you for checking out the site, and I hope it helps you out.
